Concepts:

Higher-Order Function array methods:
  • forEach() – Executes a function for each array element
  • map() – Creates a new array by applying a function to each element
  • filter() – Returns a new array with elements that match a condition
  • slice() – Returns a portion of an array without modifying it
  • find() – Returns the first element that matches a condition
    • Method Modifies Original? Returns New Array? Purpose
    forEach() ❌ No ❌ No Runs a function for each element
    map() ❌ No ✅ Yes Transforms each element
    filter() ❌ No ✅ Yes Returns elements that match a condition
    slice() ❌ No ✅ Yes Returns a portion of an array
    find() ❌ No ❌ No (returns element) Returns the first matching element
