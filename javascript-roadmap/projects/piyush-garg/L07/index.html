Concepts:

  • arrow functions defination
  • arrow functions variations
  • differences between arrow and normal functions based on:
    • this keyword
      • Feature Normal Function (function myFunc()) Arrow Function (const myFunc = () => {})
      How this is determined? Determined by how the function is called Lexically inherited (from surrounding scope)
      Value of this in global scope? window (in browsers) or global (in Node.js) Same as normal function (window/global)
      Value of this inside an object method? Refers to the object the method belongs to Refers to the outer scope, NOT the object
      Value of this inside a class method? Refers to the instance of the class Undefined (does not bind this to the instance)
    • arguments keywords
      • Feature Normal Function (function myFunc()) Arrow Function (const myFunc = () => {})
      Has arguments object? ✅ Yes (Contains all passed arguments) ❌ No (Throws ReferenceError if used directly)
      How to access arguments? Using arguments keyword inside the function Use **rest parameters** (...args) instead
      Scope of arguments Function-scoped (Local to the function) Not available (Inherits from surrounding function if used inside one)
    • hoistng
      • Feature Normal Function (function declaration) Arrow Function (const myFunc = () => {})
      Hoisted? ✅ Yes (Fully Hoisted) ❌ No (Only variable is hoisted, not the function definition)
      Can be used before declaration? ✅ Yes ❌ No (Throws TypeError)
      Stored in Memory during the compilation phase Heap, as part of variable assignment at runtime
