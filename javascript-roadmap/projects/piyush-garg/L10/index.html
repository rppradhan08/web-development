1. What is the DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the page as a tree structure where:

  • JavaScript can access and manipulate these nodes dynamically.
  • Each HTML element is a node in the tree.

    • Selecting Elements in the DOM

  • Using querySelector(): The querySelector() method allows you to select the first matching element using CSS selectors. document.querySelector("CSS_SELECTOR");
    • It returns only the first element that matches the selector.
    • If no element is found, it returns null.
  • Using The querySelectorAll(): This method selects all matching elements and returns a NodeList (not an array).
    • It returns a NodeList, which is similar to an array but does not have array methods like map() directly.
    • You can loop through the NodeList using forEach().
