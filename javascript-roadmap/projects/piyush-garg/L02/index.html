Lecture - 02

Concepts:

  • Types of variables in javascript:
    • var => Variable | Scope => GLOBAL | Reassign => Yes
    • let => literal | Scope => LOCAL | Reassign => Yes
    • const => constant | Scope => LOCAL | Reassign => No
  • Strongly typed languages:
    Means that once a variable is declared to be of a certain type, it cannot be used as another type without explicit conversion. eg: C++,C,java,etc.
  • Loosely Typed Languages:
    Variables can be used as different types without explicit conversion, which can lead to unexpected behavior. eg: python, javascript,etc.
  • A variable can only be declared once within a given scope. In Nested scope, local variable is given the priority.
