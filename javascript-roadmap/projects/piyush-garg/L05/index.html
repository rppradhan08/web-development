Lecture - 05

Concepts:

  • for loop - The for loop repeats a block of code a specific number of times.
  • while loop - The while loop continues to execute a block of code as long as a specified condition is true.
  • do while loop - The do...while loop is similar to the while loop, but it guarantees that the code block is executed at least once.
  • for...in loop - The for...in loop iterates over the properties of an object.
  • for...of - The for...of loop iterates over the values of an iterable object, such as an array.
