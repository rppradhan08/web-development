Lecture - 08

Concepts:

  • Higher-Order Function (HOF) is a function that either:
    • Takes another function as an argument.
    • Returns a function.
  • Javascript Build-in HOF map(), reduce(), filter()
    • map() – Applies a function to every element in an array.
    • filter() – Filters elements based on a condition.
    • reduce() – Reduces an array to a single value.
  • Callback function is a function passed as an argument to another function. Callbacks allow asynchronous execution in JavaScript (important in event handling & APIs).
